## Getting Started

1. Get Node.JS local environment set up with npm:

2. Install dependencies:

        $ npm install

3. Start app:

        $ npm start

4. Visit app:

		http://localhost:8080/

## Deploy to Heroku
1. heroku login
2. heroku git:remote -a hackernews-clone-restful-api
3. git push heroku develop:master
